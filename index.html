Welcome

On this site you can browse which movies and shows I have watched with my opinion on them. You can provide your own opinion as well, and suggest other movies and shows to me. This site uses bootstrap and AngularJS. The Google Apps Script Execution API to gain access to a Google spreadsheet with all the information. The omdb API is used to draw information about the movies and shows, including images. Have fun browsing the site!
